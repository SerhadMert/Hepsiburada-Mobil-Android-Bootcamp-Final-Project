# HepsiburadaMobilAndroidBootcampFinalProject
This project is a sample app for Android using iTunesApi

##Technologies
-Room DB
-Retrofit
-MVVM
-ViewBinding
-Kotlin Coroutines
-Navigation Component
-Safe-Args
-Glide
-Lottie
-RecyclerView(Infinite scroll for pagination)
